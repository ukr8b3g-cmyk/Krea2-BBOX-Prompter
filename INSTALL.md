# Install

1. Stop ComfyUI.
2. Extract or clone this repository into ComfyUI custom nodes:

```text
ComfyUI/custom_nodes/Krea2-BBOX-Prompter
```

or, if using the ZIP folder name:

```text
ComfyUI/custom_nodes/Krea2-BBOX-Prompter-Suite
```

3. Start ComfyUI.
4. Hard refresh the browser:

```text
Ctrl + F5
```

5. Add the nodes:

```text
📐 Krea2 BBOX Canvas
📝 Krea2 BBOX Prompter
📦 Krea2 BBOX Export
✨ Krea2 BBOX Prompt Effect
```

Use `✨ Krea2 BBOX Prompt Effect`, not the older `✨ Krea2 Prompt Effect` from other Krea2 suites.
